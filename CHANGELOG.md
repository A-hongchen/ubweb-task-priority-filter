# Changelog

本文件记录 `UBWEB 任务备注优先级` 脚本的主要更新。

## [1.0.1] - 2026-07-07

### Changed

- 备注已存在 `【优先级：X】` 时，会核对 X 是否等于编辑页当前优先级。
- 如果已有优先级标记正确，则跳过该任务。
- 如果已有优先级标记不正确，则替换为编辑页当前优先级。
- 更新确认弹窗和 README 文案，说明“正确跳过、错误修正”的行为。

## [1.0.0] - 2026-07-06

### Added

- 新增独立 Tampermonkey 脚本 `ubweb-task-priority-filter.user.js`。
- 在 UBWEB 任务页添加 `添加优先级` 按钮。
- 读取当前登录用户认领任务的编辑页 `优先级`。
- 将 `【优先级：X】` 写入编辑页底部 `备注` 开头。
- 已存在 `【优先级：1】` 到 `【优先级：6】` 格式前缀的任务会跳过。
- 新增 README，说明安装、更新和使用方式。
